## Project Overview

MealBridge is a digital platform designed to connect food donors (restaurants and individuals) with beneficiaries (charities and volunteers).
The goal is to reduce food waste by making the donation process faster, safer, and easier.

This repository includes the system requirements, design documents, diagrams, and all related project deliverables.
